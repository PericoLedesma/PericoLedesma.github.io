Method To solve our riddle, we will first explain the riddle for one light bulb, then increase the number of light bulbs to two or more light bulbs if possible, and then see how the problem works if we increase the number of light bulbs higher than the number of prisoners. We can see in the literature review that the 100 prisoners and one light bulb riddle is solved by choosing one person named the counter, after which every person who enters the interrogation room can turn on the light bulb just once. If the light is already switched on, they do nothing. Only when the light is switched off, and the prisoner did not yet turn the light on, turn it on. If the only person who turns off the light is the counter, he can add one to the count every time he turns it off. Furthermore, in the literature review, we can see that the solution for the problem from two prisoners on wards stays the same. Therefore, we will use an arbitrary number of prisoners in our explanation/examples to solve the riddle that is lower than 100 but higher than two, so that explainability is guaranteed. One light bulb Before we can explain the first solution to the one light bulb problem we need to model what the knowledge, and relations are of the prisoners. Such that we can see what every prisoner thinks are the possible states. Furthermore, this helps us see if we some prisoner can already deduce if everybody has been interrogated without using any protocol. We use the Kripke semantics. First we must state the assumptions we are making in the riddle. These are: • All prisoners wish to escape • All prisoners are aware of the strategy set out in protocol 1 • All prisoners are aware of their role (as counter or not) • 1 prisoner enters the room each day • The lightbulb’s initial state is either on or off • The initial state of the light bulb is known to the counter 2 These assumptions can be expressed as common knowledge. It is not necessary that the identity of the counter is common knowledge, simply that each prisoner knows their own role. It is also not necessary that one prisoner enters each day, the protocol also hold when some days no prisoner enters the room. This assumption is only important as we use it when making examples. The counter does need to know the initial state of the light bulb for this protocol to work. If the initial state is unknown to the counter the protocol fails. If the counter knows the initial state the protocol can work for either the initial state containing a lightbulb that is on or off.
